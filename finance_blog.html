The Big One

Stocks

Prop Trading Companies. They're every CS undergrad's dream internship. To land an interview with one, you need to stand apart from the average applicant. I thought that there would be no better way to do that than to create an app which is centered around stocks and finance.

03-15-2020: Created the react app and implemented worldtradingdata's API. This allows me to access any and all current stocks. To test, I allow the user to enter in a stocks symbol and it outputs that stocks information.

03-16-2020: One implementation of the api is the ability to perform currency conversions. I was very busy today so I quickly implemented that and called it a day.

03-17-2020: I want to be able to find the correlation between two stocks in a given time. If the user supplies two stock's symbols and a period of time (in days), I am able to return to them the correlatino between the two stocks over the past x days. This is based off of the starting value of the stocks on each day.

03-18-2020: I am putting off adding an styling to the app. I hate CSS. In the meantime I added user authentication using firebase. I display errors if the user enters in the wrong information.

03-19-2020: I have never been more enraged in my life. I want to append to an array field in my firestore database. However, there is an error that I simply cannot fix despite many hours of work. Not to be a drama queen but I might drop out of CS.

04-08-2020: Since my last post, classes have been moved to a virtual setting due to the outbreak of COVID-19. With regards to this application, there is good news and bad news. The good news is that I resolved the issue above. Users can now add stocks and info to a watchlist, specific to the user. The downside is that the API that I am using to grab stock info, WorldTradingData, is shutting down and I will need to find a replacement and refactor the code.
