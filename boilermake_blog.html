Boilermake 7 Project

Purdue University, 2020

Motivation

I am frustrating by the fact that spotify and applie music do not have one commone UI. I decided I wanted to create a solution to that problem using the API's provided by Spotify and Apple Music.

Hacking

We began as a team of four. After the first hour, we were down to three, and after another half day we were down to two. I created the UI as my friend Jackson worked to create user authentication for Spotify by following a youtube tutorial. After he had finished this, we had access to methods which would access both user data as well as Spotify information such as songs, artists, etc. I worked to create the functions which would provide the information necessary upon request. To detect the return types, I needed to look at the network responses to function calls Sadly, due to time constraints, I was not able to connect the UI i had created with the servers.

Technologies

The server authentication was accomplished using nodejs. Our project was created with react and javascript. Neither I nor my partner had ever used these technologies before, which explains why time had become a factor at that stage. I plan to finish applying the UI to our project, as well as introduce the apple music API. All told, the hackathon was a lot of fun, but I need to catch up on some sleep...
